Unzipping a Game (Windows)

  1. Find the downloaded Folder
  2. Right click it and select 'Extract All...'
  3. Click 'Ok' and Open the new folder that was created
  4. Find the file called '[insert-name-of-game-here].exe' and open it to run the game
  5. If there is a menu opened saying 'windows protected your PC, click on 'more info', and then 'run anyway'
