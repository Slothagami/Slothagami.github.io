Installation Instructions:

First, if its not already, Install Python 3.9.1

Make sure to select the 'add Python to PATH' checkbox when the installer opens or it won't work.

Then press Install Now



Once it's finished installing, Extract the downloaded files.

Run the setup file (if there is one) to install all the nescicary things to run the program.

Then You're finished, you can now open the main file.
