Discord Formatting Helper

Uses Hotstrings to make Discord formatting easier to use and remember. After extracting the program from the zip folder and running it:

Type To Get
c[ Inline Code
cb[ Code Block
i[ Italics
b[ Bold
bi[ Bold Italics
u[ Underline
ui[ Underline Italics
ub[ Underline Bold
ubi[ Underline Bold Italics
s[ Strikethrough
q[ Single Line Quote
bq[ Blockquote (Multi-line)
