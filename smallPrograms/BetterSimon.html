Better SIMON

Uses Python to Generate a better looking version of SIMON


If you need help installing this program, ask me on Discord.

Installation Instructions:

First, if its not already, Install Python 3.9.1
Make sure to select the 'add Python to PATH' checkbox when the installer opens or it won't work.
Then press Install Now



Then, Extract the downloaded files. Run the setup file to install all the nescicary things to run the program.

Using the Program

Open the file called BetterSimon, it will ask for your SIMON login details, it needs them to check your timetable every day.

Leave the Program Open, It can't give you your timetable if it's not running!
I suggest moving the proram to a new desktop (Press Windows + Tab to manage desktops) so that it's out of the way as you go about your buiseness.

Changing the Background

To change the background, go to the New Timetable folder, and replace background.png with the image you want for the background.

Right now, the background has to be a .png image, and it MUST be called background.png to work.
