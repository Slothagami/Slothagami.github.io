Better SIMON

  • Drastically Simplifies the timetable
  • Customiseable background.
  • Automatically detect Sport classes, highlight them and changes to an alternate background to help you know what uniform to wear.
  • The exact same timetable as on SIMON
  • Click on the date to go to the original SIMON
  • Hover over a class to see the Teachers name and Room the class is in.
  • Tells you if you need to update it.

If you need help installing this program, ask me on Discord.

Installation Instructions Here

Using the Program

Open the file called BetterSimon, enter your SIMON Login, and Leave the Program Open.

I suggest moving the proram to a new desktop (Press Windows + Tab to manage desktops)

Changing the Background

Go to the New Timetable folder, and replace background.png with the new background.

Right now, the background has to be a .png, and it MUST be called background.
