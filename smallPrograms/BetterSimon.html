Better SIMON

  • Simplifies the timetable.
  • Customiseable background.
  • Changes the background on sport days.
  • Hover over a class to see the Teachers name and Room the class is in.

Installation

Download the file, right click it and select 'Extract all'

Once the file has been downloaded, on edge, Press Alt + F and select Extensions

Then, in the bottom corner, Turn on Developer mode and click on Load Unpacked then select the folder called BetterSimon-2.1
