Better SIMON

A Browser Extension to make it easier to read your timetable.

  • Makes the timetable easier to read
  • Customiseable background
  • Hover over a class to see the Teachers name and Room the class is in

How to install

Download the file, right click it and select 'Extract all'. Once the file has been downloaded, go to your browsers extension settings and turn developer mode on. Then, find an option called "Load Unpacked" then select the folder you extracted from the zip (it should be called BetterSimon-2.x)
